# e-shop E-Commerce Website
An academic E-Shop project developed to demonstrate modern web development and e-commerce system functionalities.

## Project Overview

This project is a full-stack E-Commerce web application developed as an academic project. The system is designed to provide an online shopping platform where users can browse products, manage shopping carts, and place orders efficiently.

The project demonstrates practical implementation of web development technologies, database management, and e-commerce functionalities in a real-world business scenario.


## Features

* User Registration and Login
* Product Listing and Categories
* Shopping Cart Management
* Order Processing System
* Admin Panel for Product Management
* Database Integration
* Responsive User Interface


## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL


## Project Objectives

* To develop a functional online shopping platform
* To understand the implementation of e-commerce systems
* To apply database management and web development concepts
* To improve user interaction and online business functionality
